A **Python-based Memory Management Simulator** that demonstrates how operating systems manage memory dynamically using a **linked list data structure**. This project is designed for **Data Structures & Algorithms** coursework and academic learning.

---

## üìå Project Overview

The Memory Management Simulator models **dynamic memory allocation and deallocation** similar to an operating system. Each memory block is represented as a **linked list node**, allowing efficient tracking of allocated and free memory blocks.

The simulator helps users understand:
- How memory is allocated to processes
- How memory is freed and reused
- How fragmentation is handled by merging free blocks
